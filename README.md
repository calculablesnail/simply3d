# simply3d
Simply3D: A work in progress.

When I need to type out a quick note or an easy Python script, I use Notepad. When I come up with a melody in the middle of the night, I whistle it into a voice recorder or tap out the notes on a keyboard before moving to ModPlug. Aseprite's great when I know what I want to do, but when I'm still working that out, I tinker with it in a huge scratchpad in Paint. I bet there are a lot of other people out there like me, so I'm making something like that for working in 3D.

Simply3D will launch quickly, run smoothly, and stick with the essentials to stay that way. Its intuitive controls will make it easy for beginners to get started while giving experienced artists the tools they need to make a mock-up they can export without any fuss. Because making simple things should be simple.

Code will be made available when enough features have been completed, and I've started compiling in Windows.
